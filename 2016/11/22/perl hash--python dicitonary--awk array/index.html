perl hash -- python dictionary -- awk arrary

1.perl hash

1.1perl中出现dollar符号和{}，则说明正在使用hash array

key==> value


举个栗子：如果一个arrary名为family_name，最简单的给其赋值
key是fred value是’filinstone’
需要注意的是，key大多数时候（指不含特殊符号，如”+”来使得perl误会的时候），其外侧的单引号可以省略，但是value的单引号永远不能省略！！
$family_name{‘fred’} = ‘flintstone’;

1.2Why use hash?

Well, the general idea is that you’ll have one set of data “related to” another set of data.
In fact, if your task description includes phrases like “finding duplicates,” “unique,” “cross-reference,” or “lookup table,” it’s likely that a hash will be useful in the implementation.

1.3Hash Element Access

To access an element of a hash, you use syntax that looks like this:
$hash{$some_key}
天冷了，举一个栗子怎么够？来，举好多好多个栗子！
①
$foo = ‘bar’;
print $family_name{ $foo . ‘ney’ }; # prints ‘rubble’
说明hash能够利用“.”号连接而成的字符串（perl中“.”代表字符串连接符）
②
$family_name{‘fred’} = ‘astaire’; # gives new value to existing element $bedrock = $family_name{‘fred’}; # gets ‘astaire’; old value is lost
新值覆盖旧值
③
$granite = $family_name{‘larry’}; # No larry here: undef
$family_name中没有larry，因此$granite=undef

1.4The Hash as a Whole

%some_hash = (‘foo’, 35, ‘bar’, 12.4, 2.5, ‘hello’, ‘wilma’, 1.72e30, ‘betty’, “bye\n”);
赋值的时候元素个数必须是偶数！因为hash就是key==>value对。否则就会“unwinding the hash”
反转key和value
my %inverse_hash = reverse %any_hash;
(key, value, key, value, key, value, …).=====>> (value, key, value, key, value, key, …).
聪明的你可能已经发现了，这仅限于原hash中value也唯一的情况下
否则reverse后的hash就出错了~

1.5 Big Arrow and Hash Function

当我们看到以下这个hash的时候，一眼很难判断”2.5”是key还是value，而需要数一数”key,value,key,value,key，value…”这就很尴尬了，不过我们的Larry大神也想到了这个问题，最终用”==>”来解决这个问题
%some_hash = (‘foo’, 35, ‘bar’, 12.4, 2.5, ‘hello’, ‘wilma’, 1.72e30, ‘betty’, “bye\n”)

my %last_name = ( # a hash may be a lexical variable
‘fred’ => ‘flintstone’,
‘dino’ => undef,
‘barney’ => ‘rubble’,
‘betty’ => ‘rubble’, );
细心的你可能发现了，还是有逗号，这些逗号都是”无害而相当方便的”~
Hash Function:
①key&value Function
my %hash = (‘a’ => 1, ‘b’ => 2, ‘c’ => 3);
my @k = keys %hash;
my @v = values %hash;
顺序不一定和原hash数组中的顺序相同，但key和value的顺序一一对应
②exit function
if (exists $books{“dino”})
{ print “Hey, there’s a library card for dino!\n”;
}

③delet function
my $person = “betty”;
delete $books{$person}; # Revoke the library card for $person
