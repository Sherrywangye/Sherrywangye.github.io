一．安装circos：

1.从官网下载circos-0.68.tar（此时文件在Downloads中）

2.在Downloads中建立文件夹，名为circos，将circos-0.68.tar移至circos文件夹中并按以下命令解压缩（首先要切换目录到Downloads/circos）：

$ tar xvf circos-0.68.tar

3.生成链接

ln -s circos-0.68 current
ls

4.将circos写入环境变量，使用vi编辑器在.bashrc文件后加上circos所在的PATH

sudo su
vim ~./.bashrc

按i键进入插入模式，在最后加上一行：export PATH=~/Downloads/circos/current/bin:$PATH（PATH 一定要大写）

完成后输入”：wq”，这时已存盘退出

5.为了使对于bashrc的更改可以不重启立即生效，执行：

source .bashrc

6.检查circos是否已经在环境变量的path中

echo $PATH

7.输入/bin/env

如果出现-bash: /bin/env: No such file or directory
则：
sudo su
cd /bin
ln -s /usr/bin/env env
即可

8.这时候，在任意目录下键入：circos，都会进入circos页面，否则需进行之前的步骤

二.安装模块

1.列出需要安装的模块（这里的”需要“是总的需要，因此依然会显示已经安装了的模块）

./list.modules

2.安装所需要的模块

perl -MCPAN -e shell
>capn[1]:install Config::General
>capn[2]:install ...

根据提示逐个安装，将”…”处替换为以下常用的Perl模块

  • Config::General (v2.50 or later)
  • Font::TTF
  • GD
  • List::MoreUtils
  • Math::Bezier
  • Math::Round
  • Math::VecStat
  • Params::Validate
  • Regexp::Common
  • Set::IntSpan (v1.16 or later)
  • Text::Format

3.检查模块是否正确安装，这时，显示ok的表明已经安装好了

circos -modules

三.使用circos

1.建立配置文件

在circos/current/ect中手动建立三个配置文件：

circos.conf ideogram.conf ticks.conf

circos.conf+ideogram.conf+ticks.conf三个文件在ect中

2.写好这3个配置文件和配置文件所需要调用的文件：例如染色体组型文件xx.karyotype.txt

3.需要画图时，运行：

circos -conf ./etc/circos.conf

注意：include会从circos所在的目录开始寻找，而不是当前目录!

可以从Ensemble plants—Downloads 进行筛选下载GTF或者GFF3格式的文件（注意：不仅物种相同，这个文件需要与你之前下载的参考基因组序号一致），然后通过awk和sed制作成circos需要的格式：
