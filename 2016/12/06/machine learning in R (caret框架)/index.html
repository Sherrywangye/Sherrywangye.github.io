    1.几点认识

  • It’s useful to set an overall seed when using R,eg:

Setting each resample is useful for parallel fits,but not neccessary

  • table就是统计元素个数的一个函数,二维的table是两者信息的结合,第一个元素标示会形成列，而第二个元素标示形成行，与该行该列对应的是同时满足二者的个数，eg:

  • table可以是一维的，也刚好可以用其二维/length(matrix$y)来显示tp、tn、fp、fn

  • fit返回的是训练过后的各个参数(也就是parameters)
  • data.frame(列名=value)可以直接形成dataframe !!A$b=C，可以直接通过这种方式，给data.frame A增加一列，且列名为C
  • Training and testing set must be processed in the same way!!!
  • 模型只能用training set 来训练，validation set 和 testing set 可以预测，也可以计算error，但是不能再修正模型
  • 生成dummy variable,格式：dummyVars(b~a,data=d)
